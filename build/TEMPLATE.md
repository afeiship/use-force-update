## usage
1. import css
  ```scss
  @import "~@jswork/use-force-update/dist/style.css";

  // or use sass
  @import "~@jswork/use-force-update/dist/style.scss";

  // customize your styles:
  $use-force-update-options: ()
  ```
2. import js
